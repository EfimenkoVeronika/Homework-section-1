# Ефименко Вероника, домашнее задание.

### Первое задание: текстовый квест.

Сюжет: вор встречает подельника у банка, 
1. Подельник у него спрашивает, готов ли он к краже, ответы да-нет.
2. Спрашивает, как они попадут в здание банка: через запасной вход (1), через окно (2), через вентиляцию (3).

Взлом сейфа: угадываем число от 1 до 20, правильный код выпадает рандомно. Угадал, кража прошла успещно.
